# Blinkit Data Analysis

## Overview
This repository contains an analysis of Blinkit data, covering data processing, querying, and visualization to extract key insights. The project aims to understand trends in customer behavior, order patterns, and operational efficiency.

## Repository Contents
- **CSV File**: Raw data containing order details and customer interactions.
- **SQL Files**: Queries for data cleaning, transformation, and analysis.
- **Power BI File**: An interactive dashboard displaying insights from the data.

## Technologies Used
- **SQL**: Data processing and querying.
- **Power BI**: Data visualization and dashboard creation.

## Steps Followed
1. **Data Collection**: Gathered order data in CSV format.
2. **Data Cleaning**: Processed data using SQL queries.
3. **Exploratory Data Analysis (EDA)**: Analyzed customer behavior and order trends.
4. **Data Visualization**: Built Power BI dashboards to represent insights.

## How to Use
1. Clone this repository:
   ```sh
   git clone https://github.com/Sapna-Singh10/Blinkit.git
   ```
2. Use the SQL files to process the data in a database environment.
3. Load the Power BI file to explore visualizations and insights.

## Insights Gained
- Order frequency and peak hours.
- Customer purchase behavior and preferences.
- Revenue trends and demand fluctuations.
